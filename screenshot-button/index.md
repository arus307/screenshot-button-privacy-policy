# プライバシーポリシー / Privacy Policy

**スクショボタンV**（以下、「本拡張機能」）における利用者情報の取り扱いについて、以下のとおり定めます。

## 1. 情報の収集について
本拡張機能は、機能の改善や不具合の修正を目的として、Google社が提供するアクセス解析ツール「Google Analytics」を使用しています。
Google Analyticsはデータの収集のためにCookieを使用しますが、このデータは匿名で収集されており、**個人を特定する情報は一切含まれません**。

具体的には以下の情報を収集する場合があります：
- スクショ撮影時の以下情報
  - 利用中のプラットフォーム(配信サイト種別)
  - 配信(動画)タイトル
  - URL
- エラーログやクラッシュ情報
- 基本的なデバイス情報（OS、ブラウザのバージョンなど）

## 2. 情報の利用目的
収集した情報は、以下の目的のみに利用されます：
- 新機能の開発や既存機能の改善のため
- ユーザー体験の向上のため
- 技術的な問題の発見と修正のため

## 3. 第三者への提供
本拡張機能は、法令に基づく場合を除き、収集した情報を第三者に提供することはありません。

## 4. Google Analyticsについて

# Privacy Policy (English)

This Privacy Policy describes how **Screenshot Button V** (hereinafter referred to as "this Extension") handles user information.

## 1. Information Collection
This Extension uses "Google Analytics," an access analysis tool provided by Google, to improve features and fix bugs.
Google Analytics uses cookies to collect data, but this data is collected anonymously and **does not contain any personally identifiable information**.

Specifically, we may collect the following information:
- Information when taking a screenshot:
  - Platform in use (Type of streaming site)
  - Stream/Video title
  - URL of the page
- Error logs and crash reports
- Basic device information (OS, browser version, etc.)

## 2. Purpose of Use
The collected information is used solely for the following purposes:
- To develop new features and improve existing ones.
- To enhance the user experience.
- To identify and fix technical issues.

## 3. Third-Party Disclosure
We do not share collected information with third parties, except as required by law.
