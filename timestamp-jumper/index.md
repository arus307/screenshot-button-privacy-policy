# プライバシーポリシー / Privacy Policy

**タイムスタンプジャンパー**（以下、「本拡張機能」）における利用者情報の取り扱いについて、以下のとおり定めます。

## プライバシーとセキュリティ

- タイムスタンプデータはすべてローカル（chrome.storage.local）に保存
- 外部サーバーへのデータ送信は一切なし
- 必要最小限の権限のみを使用

---

# Privacy Policy (English)

This Privacy Policy describes how **Timestamp Jumper** (hereinafter referred to as "this Extension") handles user information.

## Privacy and Security

- All timestamp data is stored locally (chrome.storage.local)
- No data transmission to external servers
- Uses only the minimum necessary permissions
